
# AI Infographic Lesson Generator

This Streamlit app generates:
1. Lesson Notes Infographic
2. Student Exercises Infographic

The app uses:
- Streamlit (frontend)
- OpenAI Image Generation API
- GitHub (for deployment/version control)

---

## Features

✅ Colourful infographic generation  
✅ Bloom's Taxonomy exercises  
✅ Lesson notes poster  
✅ Educational worksheet design  
✅ Subject & grade customization  
✅ Multiple styles  

---

## Installation

```bash
pip install -r requirements.txt
```

Run the app:

```bash
streamlit run app.py
```

---

## Deployment on Streamlit Cloud

1. Push files to GitHub
2. Open Streamlit Cloud
3. Connect your GitHub repository
4. Deploy the app
