# Lesson Infographic Generator

A Streamlit app that generates two colorful educational infographic images from a lesson objective:

1. Notes infographic
2. Individual exercises from easy to challenging using Bloom's Taxonomy

No API is required. The images are created locally using Python and Pillow.

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## GitHub + Streamlit Cloud Deployment

1. Create a new GitHub repository.
2. Upload `app.py`, `requirements.txt`, and this `README.md`.
3. Go to Streamlit Community Cloud.
4. Connect your GitHub repository.
5. Set the main file path to:

```text
app.py
```

6. Deploy.
